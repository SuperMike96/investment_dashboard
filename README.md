# 财富收益看板

一款面向个人理财记录的深色科技感收益看板，支持多笔理财录入、收益分析、封闭期追踪和本地数据管理。

## 快速开始

```bash
npm install
npm run dev
```

打开终端输出的本地地址即可访问。

## 主要功能

- 总投入、当前总收益、总收益率和年化收益率看板
- XIRR 非定期现金流年化计算，无法收敛时使用资金加权估算
- 30 天、90 天、1 年和全部区间的资产趋势图
- 理财记录增删改、收益/亏损/封闭中筛选
- 可选封闭期，自动显示剩余天数和预计解锁日期
- 隐私模式，隐藏金额和收益率
- localStorage 本地保存、JSON/CSV 导入、JSON/CSV 导出

## 数据说明

“当前暂时盈利”表示截至当前日期的累计浮动收益金额，可以为负数。封闭期只用于追踪解锁时间，不会改变收益和 XIRR 计算结果。
